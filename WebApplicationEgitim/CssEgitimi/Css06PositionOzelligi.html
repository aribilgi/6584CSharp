Css Position Özelliği

Css position özelliği html elementlerini ekranda konumlandırmak için kullanılır.

5 Farklı Konum Özelliği Vardır

  1. static : varsayılan değerdir konumlandırmadan etkilenmez
  2. relative : göreceli
  3. fixed : sabit
  4. absolute : mutlak
  5. sticky : yapışkan

Position : relative

position:relative olan öğe üst(top), sağ(right), alt(bottom) ve sol(left) değerlerine göre konumlanır

position:relative elementi kullanımı

position: fixed (sabit)

Bir elemana position: fixed özelliği tanımlanırsa bu öğe hep aynı yerinde kalır.

Öğeyi konumlandırmak için üst(top), sağ(right), alt(bottom) ve sol(left) özellikler kullanılır.

position:fixed uygulanmış div

position:absolute-mutlak konum

absolute özelliği ile elementlere mutlaka bulunmaları gereken konumunu belirleyebiliriz.

Kapsayıcı div e position:relative özelliği veriyoruz
