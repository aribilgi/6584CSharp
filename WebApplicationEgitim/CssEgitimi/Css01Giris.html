CSS Giriş

Css Nedir?

CSS in açılımı cascading style sheets dir. Ekrandaki HTML öğelerinin tasarımını yapmamızı sağlar. 1 tane css dosyasıyla tüm uygulamayı dizayn edebildiğimiz gibi 1 den çok stil dosyası da kullanabiliriz.

Not : Herhangi bir Css uygulamazsak her tarayacı kendi css kodlarıyla html elementlerimizi şekillendirir. Dolayısıyla chrome da düzgün görünen bir tasarım firefox tarayıcıda farklı görünebilir! Buna tarayıcı uyumsuzluğu denir. Bu sorunu çözmek için genelde bir css dosyasıyla tüm elementlerin css leri sıfırlanıp sonrasında kendi kurallarımızı ekleriz.

Ayrıca uygulamalarda gerektiğinde harici bir eklenti kullanmamız gerekebilir. Kullandığımız eklentinin de kendine ait tasarım dosyası olabilir bunu da projeye dahil ederek kullanırız.

CSS Yazma Seçenekleri

  1. Inline yani direk html elementinin içine style attribute ü ile yazabiliriz.
  2. Sayfada head kodları arasında style tag larının arasında yazabiliriz.
  3. Harici bir dosya oluşturup (stil.css gibi) bu dosyayı yine sayfanın head alanında çağırıp tüm syfaları 1 dosya ile dizayn edebiliriz.

Css uygulamak için öncelikle css uygulayacağımız element ya da elementleri seçip sonrasında css kodları ile bu elementlere css uygulayabiliriz.

Inline Css Yazımı

Elemetin başlangıç tag leri arasına style="" diyerek css yazabiliriz. Eğer 1 den çok css özelliği ekleyeceksek aralarına ; ekleyerek yazarız.

Harici CSS Dosyası Kullanma

Projeye(veya projedeki dilediğimiz bir klasöre) sağ tıklayıp Add > Cascading style sheet menüsünden oluşacak css dosyasına dilediğimiz ismi verip ok diyerek .css uzantısıyla dosyayı oluşturuyoruz.

Oluşan Css dosyası içerisine sadece css kodları yazıyoruz.

Son olarak oluşturduğumuz css dosyasının sayfamızda çalışması için sayfanın head kodları arasına dosyayı sürükle bırak yöntemiyle veya elle link kodu yazarak dahil etmeliyiz aksi taktirde yazacağımız css kodları sayfaya işlemez!

Not : harici css kullandığımızda bazen yaptığımız css değişiklikleri tarayıcıda uygulanmaz bunun sebebi tarayıcıların css gibi statik dosyaları ön belleklerine almalarıdır! Bu durumda klavyeden ctrl tuşuna basılı tutarak sayfayı refresh yaparsak css dosyası yeniden kaynağından çekilir ve ön bellek boşaltılarak bu yeni dosya kullanılır.

CSS Seçiciler

1-Element Seçici

Sayfadaki HTML elementleri seçerek bu elementlere css uygulayabiliriz. Örneğin body elementi veya div, p, h1-h6, ul vb gibi.

2-Id Seçici

Elementlere sonradan tanımlayacağımız id attribute ü ile ulaşıp css kodu uygulayabiliriz. Bir elemente id si ile ulaşmak için css yazdığımız yerde # işareti koyup id ye yazdığımız değeri belirtiriz, şöyle; #idSecici{ buraya css kodları yazılır }

3-Class Seçici

Css class seçici kullanabilmek için de her hangi bir html elementinin başlangıç etiket isminden sonra 1 boşluk bırakıp class="" şeklinde tanımlama yaparız. İstediğimz class ismini verebiliriz. Css tarafında class seçici ile kod yazmak için . seçicisi kullanılır. Style kodları arasında .classSecici{} şeklinde elementlere ulaşıp css uygulayabiliriz.

Not : Id seçici yi genellikle 1 sayfadaki sadece 1 element için kullanırız! Class seçici ise birden çok elementi seçmek için kullanılır!
