Css ile Arkaplan resmi

Html elementlerine css ile arkaplan resmi ekleyebiliriz.

Css ile Arkaplan resim tekrar ayarı

Html elementlerine css ile arkaplan resmi ekleyebiliriz. Eklediğimiz resmin tüm alanda tekrar etmesini istemezsek background-repeat:no-repeat; özelliğini eklemeliyiz.

Resmin dikeyde veya yatayda tekrar etmesini de sağlayabiliriz. Bunun için background-repeat:repeat-y; dikeyde tekrar et background-repeat:repeat-x; yatayda tekrar et demeliyiz.
