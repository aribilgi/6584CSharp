Jquery Seçiciler

jQuery, John Resig tarafından 2006 yılında geliştirilmiş ve şu an geniş bir jQuery ekibi tarafından gelişimi sürdürülen bir açık kaynak JavaScript kütüphanesidir. jQuery en popüler çapraz-platform JavaScript kütüphanesi ya da farklı bir JavaScript web geliştirme çatısıdır denebilir.

Jquery Projeye 3 Farklı Yöntemle Eklenebilir

1-CDN Sitelerinden(Jquery sitesinden veya google a Jquery cdn yazarak bulabiliriz) alınan kodu sayfaya ekleyerek

2-Nuget Paket yöneticisi üzerinden, Projeye sağ tıklayıp nuget penceresini açıp Jquery kütüphanesini seçip yükleyebiliriz.

3-Jquery sitesinden indireceğimiz js kütüphanesini projeye sürükle bırakla yükleyerek
