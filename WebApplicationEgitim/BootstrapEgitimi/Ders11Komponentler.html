Bootstrap Komponentler

Bootstrap web uygulamalarımızda kullanabileceğimiz bir çok hazır komponent sunar örneğin resimleri sırayla kaydıran bir slider yapmak için carousel komponenti vardır.

Komponentleri kullanabilmek için sayfada css kütüphanesiyle birlikte Bootstrap javascript kütüphanesini de kullanmamız gerekir.

CDN Kullanımı

Web uygulamaları internet üzerinden çalıştığı için dilersek kütüphaneleri uygulama içerisine eklemeden web üzerindeki belirli server lardan çekerek kullanabiliriz.

Bu Server lara kısaca CDN denir ve bize css, js vb kütüphaneleri kolayca kullanma imkanı sunar.
