Bootstrap Giriş

Bootstrap kütüphanesi web uygulamaları tasarlarken bize hazır bir css ve js kütüphanesi sunar. Bu sayede ekstra bir css kodu yazmadan proje geliştirmemize yardımcı olur.

Bootstrap i Yükleme

Kendi web sitesinden indirip projeye sürükle bırakla yükleyebiliriz.
CDN kütüphane server larından dosyayı çekerek kullanabiliriz.

Bootstrap Kırılım Noktaları

  1. x-small : ön ek xs kırılım : 576 px ve altı
  2. small ön ek : sm - kırılım 576px ve üstü
  3. medium : ön ek md : 768px ve üstü
  4. large : ön ek : lg: 992px ve üstü
  5. extra large : ön ek : xl : 1200px ve üstü
  6. xxl : 1400px ve üstü

Containers - Kapsayıcı

container sınıfı kullanılarak sağdan soldan boşluk bırakılan ekran tasarımı yapılabilir. container class ı yukarıda belirttiğimiz kırılımlara göre kendine width değeri belirleyip kenarlarda boşluk bırakır.
container-fluid class ını kullanarak full ekran tasarımlı çıktılar üretebiliriz. container-fluid soldan sağa ekran genişliğinin tamamını kullanır.
