alttan margin vermek için mb-0 .. 5 arasında
soldan boşluk vermek için ms-0 ..5 arasında

eğer 4 yönden boşluk vereceksek m-0 .. 5 arasında değer ile class verebiliriz.

my-5 classı uygulanan p üstten ve alttan 5 birimlik margin

Resimleri Responsive Hale Getirme

Resim img elementlerini tarayıcı genişliğine göre ayarlamak için bootstrap de img class ları vardır.
img-fluid class ı tanımlarsak resim responsive olur.
img-thumbnail class ı dışı çizgili küçük resim için kullanılabilir.

Boyutlandırma için w ile başlayan class ları da kullanabiliriz. % biriminden verilen sayı kadar genişlik uygulanır
w-25 : %25
w-50 : %50
w-75 : %75
w-100: %100
