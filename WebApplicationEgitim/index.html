Sayfada Resim Gösterme

Web sayfalarında resim göstermek için html img elementi kullanılır.
img elementine src özelliği ile resmin yolu belirtilirse ekranda resim gösterilmiş olur.

img elementine width ve height özellikleri ile genişlik ve yükseklik tanılanabilir

HTML Table Elementi Kullanımı

table elementi ana kapsayıcıdır. table içerisinde satır oluşturmak için önce tr elementi açılıp kapatılır, sütun oluşturmak için tr içine td elementi açılıp kapatılır. 1 satır 1 sütunluk tablo.
table elementinin çerçevelerini göstermek için table ye border="1 tanımlanır."
